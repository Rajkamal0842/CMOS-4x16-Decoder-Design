This project demonstrates the application of CMOS technology to create complex combinational logic, focusing on optimizing the layout and minimizing power dissipation.

🛠️ Technical Specifications
Design Style: Static CMOS Logic.
Hierarchy: Built using a modular approach (using 2x4 decoder blocks).
Technology Node: gpdk90nm.
Tools: Cadence Virtoso, LTSpice.

 🚀 Key Features
Detailed schematic design of PMOS and NMOS networks.
Analysis of Propagation Delay and Power Consumption.
Verification of the 16-output truth table through transient analysis.
